# SmartStamp-PDF (ระบบลงรับหนังสืออิเล็กทรอนิกส์) 📑✅

ระบบเว็บแอปพลิเคชันสำหรับประทับตราลงรับหนังสือราชการลงบนไฟล์ PDF (E-Stamp) พัฒนาขึ้นเพื่อช่วยลดขั้นตอนการทำงานสารบรรณในหน่วยงานราชการ ให้สามารถลงรับหนังสือได้อย่างรวดเร็วและเป็นระเบียบ

## ✨ คุณสมบัติเด่น (Features)
- 📍 **Click-to-Stamp:** เลือกจุดวางตราประทับบนหน้าเอกสารได้อิเล็กทรอนิกส์ตามต้องการ
- 📏 **Dynamic Scaling:** มีสไลเดอร์ปรับขนาดตราประทับ (50% - 150%) ให้เหมาะสมกับพื้นที่ในเอกสาร
- 🏢 **Config-Driven:** ปรับเปลี่ยนชื่อหน่วยงาน และรายชื่อกลุ่มงานได้ง่ายๆ ผ่านไฟล์ `index.php` เพียงจุดเดียว
- 🗹 **Auto-Checkbox:** แสดงเครื่องหมายติ๊กถูกหน้ากลุ่มงานเจ้าของเรื่องให้อัตโนมัติ
- 🖨️ **Print & Download:** รองรับการดาวน์โหลดไฟล์ที่ Stamp แล้ว หรือสั่งพิมพ์ออกเครื่องพิมพ์ได้ทันที
- 📱 **Responsive Design:** ใช้งานได้ทั้งบนคอมพิวเตอร์และแท็บเล็ต

## 🛠 การติดตั้ง (Installation)
1. Clone หรือ Download ไฟล์ทั้งหมดไปวางใน Web Server (รองรับ Shared Hosting ทั่วไป)
2. ตรวจสอบว่าในโฟลเดอร์มีไฟล์ฟอนต์ `THSarabunNew.ttf` เรียบร้อยแล้ว
3. เข้าใช้งานผ่าน Browser ได้ทันที

## ⚙️ การตั้งค่า (Configuration)
คุณสามารถปรับเปลี่ยนข้อมูลหน่วยงานได้ที่ส่วนบนสุดของไฟล์ `index.php` ในตัวแปร `$config`:

```php
$config = [
    "officeName" => "ชื่อหน่วยงานของคุณ",
    "developer" => [
        "name"     => "ชื่อผู้พัฒนา",
        "position" => "ตำแหน่ง"
    ],
    "groups" => [
        ["กลุ่มงาน 1", "กลุ่มงาน 2", "กลุ่มงาน 3"],
        ["กลุ่มงาน 4", "กลุ่มงาน 5", "กลุ่มงาน 6"]
    ]
];
```
## 📚 เทคโนโลยีที่ใช้ (Tech Stack)
- Frontend: HTML5, CSS3 (Flexbox), JavaScript
- Backend: PHP (สำหรับ Configuration)
- PDF Processing: pdf-lib & pdf.js
- Fonts: TH Sarabun New (SIPA Thailand)
